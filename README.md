# User Management System

Module 15 Assignment - Express.js Backend Project Structure

## Project Structure

```
src/
├── config/
├── controllers/
├── middlewares/
├── models/
├── routes/
├── services/
├── utils/
```

## Features

- Create User
- Read User
- Update User
- Delete User
- Get All Users
- Get User By ID
- Login & Logout
- Change Password
- Update Profile
- Make Admin / Remove Admin
- Search & Filter Users
- Block / Unblock Users
- Verify Email
- Upload Profile Picture
- Delete Account

## Technologies Used

- Node.js
- Express.js

## Installation

```bash
npm install
```

## Run the Project

```bash
node server.js
```

Server URL:

```text
http://localhost:3001
```
